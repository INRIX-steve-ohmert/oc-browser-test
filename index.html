Please note that "LI" here corresponds to the line items in the OpenCar browser requirements spreadsheet, for example, "Embedding custom non-visible data" corresponds to the "dataset & data-* attributes" line item. As in the spreadsheet, the individual items in green here are required, and those in blue are preferred. This series of tests checks if a browser recognizes a particular feature, it does not ascertain if the browser has implemented that feature correctly.

This browser test was adapted from Neils Leenheer's html5test (site / GitHub), as well as Lea Verou's css3test (site / GitHub). It has been adjusted to suit OpenCar's needs.
